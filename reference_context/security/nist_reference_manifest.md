# NIST Security and Privacy Reference Manifest

Advisory NIST public-reference manifest for security, privacy, supplier assurance, and technical-risk evidence questions.

This reference intelligence is advisory only. Uploaded evidence, deterministic council logic, and accountable human approval remain separate.

Tags: cybersecurity, govern, identify, protect, detect, respond, recover, security
Domains: technical_risk
### Advisory Boundary

Use as advisory reference context for security control discovery, SOC2-style evidence requests, access control checks, incident response, and recovery obligations.
