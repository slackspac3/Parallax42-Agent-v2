# Legal Reference Intelligence

Advisory case-law, citation-verification, and contract-clause context for issue spotting. This lane is not legal advice and does not approve agreements.

Typical sources:
- CourtListener REST API and citation lookup
- CourtListener or Free Law Project bulk data
- CUAD-compatible clause records
- SEC EDGAR material-contract exhibits, when separately imported

Generated import artifacts in this directory are reference intelligence only. Uploaded case evidence remains separate.
